# UX心理学コンセプト

ユーザーエクスペリエンス向上のための心理効果とデザイン原則のリファレンス。

## 概要

意思決定に影響を与える心理学とバイアスを理解し、より効果的なUXデザインを実現する。

## カテゴリ別インデックス

### 認知・知覚

| 効果 | 概要 | 活用場面 |
|-----|-----|---------|
| 認知負荷 | 情報処理にかかる精神的エネルギー | UI簡素化 |
| 選択的注意 | 目的に沿った情報への集中 | 情報設計 |
| バナー・ブラインドネス | 広告バナーの無意識的無視 | 広告配置 |
| 視覚的階層 | デザイン要素の優先順位付け | レイアウト |

### バイアス・判断

| 効果 | 概要 | 活用場面 |
|-----|-----|---------|
| アンカー効果 | 先行情報が後続評価に影響 | 価格表示 |
| 確証バイアス | 既存信念に合致する情報を優先 | コンテンツ設計 |
| 期待バイアス | 事前期待が体験評価に影響 | ブランディング |
| 親近性バイアス | 経験済みデザインへの親しみ | UI一貫性 |
| フレーミング効果 | 情報提示方法が判断に影響 | コピーライティング |

### 行動誘導

| 効果 | 概要 | 活用場面 |
|-----|-----|---------|
| ナッジ効果 | 非強制的な行動誘導 | デフォルト設定 |
| デフォルト効果 | デフォルト値への固執 | オプトイン/アウト |
| 段階的要請 | 小さな要求から大きな要求へ | オンボーディング |
| おとり効果 | 第三選択肢で特定選択を誘導 | 料金プラン |
| 損失回避 | 損失を避けたい傾向 | 限定オファー |

### モチベーション

| 効果 | 概要 | 活用場面 |
|-----|-----|---------|
| 目標勾配効果 | 目標接近で行動加速 | プログレスバー |
| 変動型報酬 | 予測不能な報酬で興味維持 | ゲーミフィケーション |
| ゲーミフィケーション | ゲーム要素でモチベーション向上 | エンゲージメント |
| ツァイガルニク効果 | 未完了タスクが記憶に残る | タスク管理 |
| 誘惑の結びつけ | 好きな活動と苦手なタスクを結合 | 習慣形成 |

### 価値知覚

| 効果 | 概要 | 活用場面 |
|-----|-----|---------|
| 美的ユーザビリティ効果 | 美しいUIは使いやすく感じる | UI美観 |
| 労働の錯覚 | 労力表示で価値向上 | 検索結果表示 |
| 希少性効果 | 限定感で価値向上 | 在庫表示 |
| 授かり効果 | 所有物への過大評価 | 試用期間 |
| 社会的証明 | 他者評価への影響 | レビュー表示 |

### 記憶・体験

| 効果 | 概要 | 活用場面 |
|-----|-----|---------|
| ピーク・エンドの法則 | ピークと終わりで体験評価 | 体験設計 |
| 系列位置効果 | 最初と最後が記憶に残る | ナビゲーション |
| プライミング効果 | 先行刺激が後続行動に影響 | 導入ページ |

### UX設計手法

| 効果 | 概要 | 活用場面 |
|-----|-----|---------|
| 段階的開示 | 情報を段階的に表示 | 複雑なフォーム |
| 意図的な壁 | 意図的な行動障壁 | 確認ダイアログ |
| 反応型オンボーディング | 行動に応じたオンボーディング | 新規ユーザー |
| 好奇心ギャップ | 情報欠如で興味喚起 | CTAボタン |
| スキューモーフィズム | 現実物の模倣デザイン | 直感的UI |

### 注意すべき効果

| 効果 | 概要 | 注意点 |
|-----|-----|--------|
| 決断疲れ | 繰り返し決断で判断力低下 | 選択肢を絞る |
| 共感ギャップ | ユーザー視点の欠如 | ユーザーテスト |
| 誘導抵抗 | 制約への心理的反発 | 押し付けを避ける |
| 観察効果 | 観察による行動変化 | テスト設計 |
| 調査バイアス | 調査結果の偏り | 質問設計 |

## 使い方

1. **課題特定**: 解決したいUX課題を明確化
2. **効果選択**: カテゴリから関連する心理効果を選択
3. **詳細確認**: [EFFECTS.md](EFFECTS.md)で詳細を確認
4. **適用**: デザインに心理効果を適用

## 詳細リファレンス

各心理効果の詳細な説明、具体例、適用時の注意点は[EFFECTS.md](EFFECTS.md)を参照。

